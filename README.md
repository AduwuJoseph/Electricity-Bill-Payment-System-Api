# Electricity-Bill-Payment-System-Api
Build a backend service for electricity bill vending and payments using an event-driven architecture.

The Api is implemented using .Net 8.0 and the architecture used is clean architecture

## Run our API project.

Step 1: Clone the project
    Url: (https://github.com/AduwuJoseph/Electricity-Bill-Payment-System-Api.git) ebps

Step 2: Open with Visual Studio 2022 or with VS Code

Step 3: Build the Project

Step 4: Run the application

## Note
To mock the AWS SNS and AWS SQS, i used the HttpContext.Session to mock how the AWS SNS and SQS operate
